## üõ† Installation & Setup  

### Prerequisites
Make sure you have a **C++ compiler** installed:
- **GCC** (Linux/MacOS)
- **Clang** (MacOS/Linux)  
- **MSVC** (Windows)
- **MinGW** (Windows)

### Compilation
To compile any C++ file, use:

```bash
# Basic compilation
g++ filename.cpp -o output_name

# With debugging symbols
g++ -g filename.cpp -o output_name

# With C++11/14/17 standard
g++ -std=c++17 filename.cpp -o output_name

# Run the compiled program
./output_name
```

### Running Examples
```bash
# Navigate to any lecture folder
cd "Lecture 99"

# Compile and run
g++ Articulation_points.cpp -o articulation
./articulation
```
